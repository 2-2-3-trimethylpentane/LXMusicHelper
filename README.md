# LXMusicHelper

本项目是用于第三方音乐平台快速跳转LX Music的一个小工具，使用 python 与 AI 辅助开发技术（vibe coding）开发。仅用于个人学习用途与 AI辅助开发 方面的研究。

**在寻找LX Music吗？（[桌面端仓库](https://github.com/lyswhut/lx-music-desktop)|[移动端仓库](https://github.com/lyswhut/lx-music-mobile)）**

> [!IMPORTANT]
> 本项目**不涉及任何音频播放、歌词/歌曲下载功能，也不包含这个功能**，对此方面的问题我们**一律不受理**（除本项目输出参数传递异常所造成的音频播放错误外）。

> [!note]
> 作者由于学业繁忙与自身精力的局限，不会经常更新项目。请不要在issue里无能狂怒，这并没有用。如果你想添加工具的功能及适用范围请自行修改后提交pr到此仓库。

## 开发手记
### 突发奇想
我个人使用过LX Music ，平时使用时总感觉官方的浏览器用户脚本只支持一个平台还是差点意思，又知道最近AI发展迅速，遂使用Google Gemini 3.0 Flash&Thinking vibe coding了一个这个项目。

### 开发过程
刚开始还蛮顺利的，某动物和某三个数字平台很快就兼容好了。之后又去兼容一些分享链接类的东西，AI就开始瞎改了。到了kg、kw这里原来正常的模块就被AI改得乱七八糟的，好不容易把kg的单曲适配做好，kw的又不行了，kw的做好了，kg的又炸了。最后发现kw的API不稳定，于是去除了kw的模块，拿了一个最稳定的版本作为1.0 release。
